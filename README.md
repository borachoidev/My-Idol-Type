# [My Idol Type Test (MITT)](https://september21st.github.io/My-Idol-Type/)

내가 만약 아이돌로 데뷔한다면 나는 어떤 아이돌일까?  
나의 아이돌 유형을 알아보는 성격 테스트!

## 서비스 소개

12개의 문항을 통해 자신의 아이돌 유형을 알 수 있습니다! 총 8가지 유형 중 당신은 어떤 유형인가요?
(테스트 결과는 재미로만 !)

## 사용 기술

- frontend : React
  백엔드 없이 프론트엔드로만 구현
